# 📖 Project Overview

Retail businesses must accurately forecast future demand to minimize inventory costs while ensuring product availability.

This project presents an **End-to-End Sales Forecasting & Demand Intelligence System** built using historical retail sales data.

The solution combines:

- Data Engineering
- Exploratory Data Analysis (EDA)
- Time Series Forecasting
- Machine Learning
- Anomaly Detection
- Customer Segmentation
- Interactive Business Dashboard

The project enables business users to make informed inventory and supply chain decisions using predictive analytics.

---

# 🎯 Project Objectives

- Forecast future sales demand
- Compare multiple forecasting models
- Detect abnormal sales behavior
- Segment customers based on purchasing patterns
- Build an interactive business dashboard
- Support inventory planning through data-driven insights

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/ACT2039/sales-forecasting-demand-intelligence-system.git
```

Move into the project

```bash
cd sales-forecasting-demand-intelligence-system
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run Streamlit

```bash
streamlit run app.py
```
